# RecoMart MLOps

RecoMart is an end-to-end recommendation-system data and ML pipeline built with
Python, FastAPI, Airflow, PostgreSQL/SQLite, S3-compatible storage, DVC, MLflow,
Pandas, PyArrow, and Scikit-Learn.
